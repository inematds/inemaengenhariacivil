# Exemplo: Sapata Isolada

## Problema

Dimensionar uma sapata isolada sob pilar de seção 30x30cm:

- **Carga normal característica:** Nk = 600 kN
- **Momento característico:** Mk = 20 kN·m (direção x)
- **Tensão admissível do solo:** σ_adm = 150 kPa (SPT médio = 8 golpes)
- **Concreto:** C25 (fck = 25 MPa)
- **Aço:** CA-50 (fyk = 500 MPa)
- **Profundidade de fundação:** 1,5 m
- **Classe agressividade:** CA-II

## Verificações Necessárias

1. Dimensões em planta (B x L)
2. Tensão máxima no solo <= σ_adm
3. Altura da sapata (rigidez)
4. Armadura de flexão nas duas direções
5. Punção
6. Cisalhamento
7. Estabilidade (FS deslizamento e tombamento)
